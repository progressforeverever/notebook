  2. JREпјҲJava Runtime EnvironmentпјүпјҡJRE жҳҜ Java иҝҗвҫҸж—¶зҺҜеўғпјҢеҢ…жӢ¬дәҶ Java иҷҡжӢҹжңә пјҲJVMпјүе’Ң Java ж ҮеҮҶзұ»еә“пјҲJava APIпјүгҖӮJRE жҳҜй’ҲеҜ№ Java еә”вҪӨзЁӢеәҸзҡ„пјҢе®ғжҸҗдҫӣдәҶеңЁи®Ў з®—жңәдёҠиҝҗвҫҸ Java еә”вҪӨзЁӢеәҸжүҖйңҖзҡ„жңҖвј©зҺҜеўғгҖӮ

  3. JVMпјҲJava Virtual MachineпјүпјҡJVM жҳҜ Java иҷҡжӢҹжңәпјҢжҳҜ Java зЁӢеәҸиҝҗвҫҸзҡ„зҺҜеўғгҖӮJVM иҙҹиҙЈе°Ҷ Java д»Јз Ғи§ЈйҮҠжҲ–зј–иҜ‘дёәжң¬ең°жңәеҷЁд»Јз ҒпјҢе№¶еңЁиҝҗвҫҸж—¶жҸҗдҫӣеҝ…иҰҒзҡ„зҺҜеўғвҪҖжҢҒпјҢвҪҗеҰӮеҶ… еӯҳз®ЎзҗҶгҖҒеһғеңҫеӣһж”¶гҖҒе®үе…ЁжҖ§зӯүгҖӮJVM зҡ„дё»иҰҒдҪңвҪӨжҳҜе°Ҷ Java д»Јз ҒиҪ¬жҚўдёәеҸҜд»ҘеңЁи®Ўз®—жңәдёҠиҝҗ вҫҸзҡ„жңәеҷЁз ҒпјҢе№¶иҙҹиҙЈзЁӢеәҸзҡ„жү§вҫҸгҖӮ

JRE жҳҜ Java иҝҗвҫҸж—¶зҺҜеўғпјҢеҢ…жӢ¬дәҶ Java иҷҡжӢҹжңәе’Ң Java ж ҮеҮҶзұ»еә“пјҢвҪӨдәҺеңЁи®Ўз®—жңәдёҠиҝҗвҫҸ Java еә”вҪӨзЁӢеәҸгҖӮ

JVM жҳҜ Java иҷҡжӢҹжңәпјҢжҳҜ Java зЁӢеәҸиҝҗвҫҸзҡ„зҺҜеўғпјҢиҙҹиҙЈе°Ҷ Java д»Јз ҒиҪ¬жҚўдёәеҸҜд»ҘеңЁи®Ўз®—жңә дёҠиҝҗвҫҸзҡ„жңәеҷЁз ҒпјҢе№¶жҸҗдҫӣеҝ…иҰҒзҡ„зҺҜеўғвҪҖжҢҒ

жҗһжё…жҘҡиҝҷдёӘй—®йўҳпјҢжҲ‘и§үеҫ—вҫёе…ҲдҪ иҰҒжё…жҘҡ Java зЁӢеәҸжҳҜеҰӮдҪ•еңЁи®Ўз®—жңәдёҠиҝҗвҫҸзҡ„гҖӮ

JVM еҸҜд»ҘзҗҶи§Јзҡ„д»Јз Ғе°ұеҸ«еҒҡеӯ—иҠӮз ҒпјҲеҚіжү©еұ•еҗҚдёә .class зҡ„ж–Үд»¶пјүпјҢе®ғдёҚйқўеҗ‘д»»дҪ•зү№е®ҡзҡ„еӨ„зҗҶеҷЁпјҢеҸӘйқўеҗ‘иҷҡжӢҹжңәгҖӮJava иҜӯиЁҖйҖҡиҝҮеӯ—иҠӮз Ғзҡ„ж–№ејҸпјҢеңЁдёҖе®ҡзЁӢеәҰдёҠи§ЈеҶідәҶдј з»ҹи§ЈйҮҠеһӢиҜӯиЁҖжү§иЎҢж•ҲзҺҮдҪҺзҡ„й—®йўҳпјҢеҗҢж—¶еҸҲдҝқз•ҷдәҶи§ЈйҮҠеһӢиҜӯиЁҖеҸҜз§»жӨҚзҡ„зү№зӮ№гҖӮ

з§»дҪҚиҝҗз®—з¬ҰжҳҜжңҖеҹәжң¬зҡ„иҝҗз®—з¬Ұд№ӢдёҖпјҢеҮ д№ҺжҜҸз§Қзј–зЁӢиҜӯиЁҖйғҪеҢ…еҗ«иҝҷдёҖиҝҗз®—з¬ҰгҖӮз§»дҪҚж“ҚдҪңдёӯпјҢиў«ж“ҚдҪңзҡ„ж•°жҚ®иў«и§ҶдёәдәҢиҝӣеҲ¶ж•°пјҢз§»дҪҚе°ұжҳҜе°Ҷе…¶еҗ‘е·ҰжҲ–еҗ‘еҸіз§»еҠЁиӢҘе№ІдҪҚзҡ„иҝҗз®—гҖӮ

# йқҷжҖҒеҸҳйҮҸжңүд»Җд№ҲдҪңз”Ёпјҹ

йқҷжҖҒеҸҳйҮҸд№ҹе°ұжҳҜиў« static е…ій”®еӯ—дҝ®йҘ°зҡ„еҸҳйҮҸгҖӮе®ғеҸҜд»Ҙиў«зұ»зҡ„жүҖжңүе®һдҫӢе…ұдә«пјҢж— и®әдёҖдёӘзұ»еҲӣе»әдәҶеӨҡе°‘дёӘеҜ№иұЎпјҢе®ғд»¬йғҪе…ұдә«еҗҢдёҖд»ҪйқҷжҖҒеҸҳйҮҸгҖӮд№ҹе°ұжҳҜиҜҙпјҢйқҷжҖҒеҸҳйҮҸеҸӘдјҡиў«еҲҶй…ҚдёҖж¬ЎеҶ…еӯҳпјҢеҚідҪҝеҲӣе»әеӨҡдёӘеҜ№иұЎпјҢиҝҷж ·еҸҜд»ҘиҠӮзңҒеҶ…еӯҳгҖӮ

йқҷжҖҒеҸҳйҮҸжҳҜйҖҡиҝҮзұ»еҗҚжқҘи®ҝй—®зҡ„пјҢдҫӢеҰӮStaticVariableExample.staticVarпјҲеҰӮжһңиў« privateе…ій”®еӯ—дҝ®йҘ°е°ұж— жі•иҝҷж ·и®ҝй—®дәҶпјүгҖӮ

# йқҷжҖҒж–№жі•дёәд»Җд№ҲдёҚиғҪи°ғз”ЁйқһйқҷжҖҒжҲҗе‘ҳ?

иҝҷдёӘйңҖиҰҒз»“еҗҲ JVM зҡ„зӣёе…ізҹҘиҜҶпјҢдё»иҰҒеҺҹеӣ еҰӮдёӢпјҡ

  1. йқҷжҖҒж–№жі•жҳҜеұһдәҺзұ»зҡ„пјҢеңЁзұ»еҠ иҪҪзҡ„ж—¶еҖҷе°ұдјҡеҲҶй…ҚеҶ…еӯҳпјҢеҸҜд»ҘйҖҡиҝҮзұ»еҗҚзӣҙжҺҘи®ҝй—®гҖӮиҖҢйқһйқҷжҖҒжҲҗе‘ҳеұһдәҺе®һдҫӢеҜ№иұЎпјҢеҸӘжңүеңЁеҜ№иұЎе®һдҫӢеҢ–д№ӢеҗҺжүҚеӯҳеңЁпјҢйңҖиҰҒйҖҡиҝҮзұ»зҡ„е®һдҫӢеҜ№иұЎеҺ»и®ҝй—®гҖӮ
  2. еңЁзұ»зҡ„йқһйқҷжҖҒжҲҗе‘ҳдёҚеӯҳеңЁзҡ„ж—¶еҖҷйқҷжҖҒж–№жі•е°ұе·Із»ҸеӯҳеңЁдәҶпјҢжӯӨж—¶и°ғз”ЁеңЁеҶ…еӯҳдёӯиҝҳдёҚеӯҳеңЁзҡ„йқһйқҷжҖҒжҲҗе‘ҳпјҢеұһдәҺйқһжі•ж“ҚдҪңгҖӮ
public class Example {
    // е®ҡд№үдёҖдёӘеӯ—з¬ҰеһӢеёёйҮҸ
    public static final char LETTER_A = 'A';

    // е®ҡд№үдёҖдёӘеӯ—з¬ҰдёІеёёйҮҸ
    public static final String GREETING_MESSAGE = "Hello, world!";

    public static void main(String[] args) {
        // иҫ“еҮәеӯ—з¬ҰеһӢеёёйҮҸзҡ„еҖј
        System.out.println("еӯ—з¬ҰеһӢеёёйҮҸзҡ„еҖјдёәпјҡ" + LETTER_A);

        // иҫ“еҮәеӯ—з¬ҰдёІеёёйҮҸзҡ„еҖј
        System.out.println("еӯ—з¬ҰдёІеёёйҮҸзҡ„еҖјдёәпјҡ" + GREETING_MESSAGE);
    }
}

# String е’Ң StringBufferгҖҒStringBuilder зҡ„еҢәеҲ«жҳҜд»Җд№Ҳпјҹ

String жҳҜдёҚеҸҜеҸҳзҡ„пјҲimmutableпјүеҜ№иұЎпјҢвҪҪ StringBuffer е’Ң StringBuilder еҲҷжҳҜеҸҜеҸҳзҡ„ пјҲmutableпјүеҜ№иұЎгҖӮ

String еҜ№иұЎвјҖж—Ұиў«еҲӣе»әпјҢе°ұдёҚеҸҜдҝ®ж”№пјҢд»»дҪ•зҡ„еӯ—з¬ҰдёІж“ҚдҪңйғҪдјҡиҝ”еӣһвјҖдёӘж–°зҡ„ String еҜ№иұЎпјҢиҝҷ еҸҜиғҪеҜјиҮҙйў‘з№Ғзҡ„еҜ№иұЎеҲӣе»әе’Ңй”ҖжҜҒпјҢеҪұе“ҚжҖ§иғҪгҖӮ

вҪҪ StringBuffer е’Ң StringBuilder е…Ғи®ёиҝӣвҫҸдҝ®ж”№ ж“ҚдҪңпјҢжҸҗдҫӣдәҶвјҖз§Қжӣҙвҫјж•Ҳзҡ„еӯ—з¬ҰдёІеӨ„зҗҶвҪ…ејҸгҖӮ

StringBuffer е’Ң StringBuilder зҡ„дё»иҰҒеҢәеҲ«еңЁдәҺзәҝзЁӢе®үе…ЁжҖ§е’ҢжҖ§иғҪвҪ…вҫҜгҖӮ

StringBuffer жҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢжүҖжңүвҪ…жі•йғҪжҳҜеҗҢжӯҘзҡ„пјҢеӣ жӯӨеҸҜд»Ҙиў«еӨҡдёӘзәҝзЁӢеҗҢж—¶и®ҝй—®е’Ңдҝ®ж”№гҖӮвҪҪ StringBuilder дёҚжҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢйҖӮвҪӨдәҺеҚ•зәҝзЁӢзҺҜеўғдёӢзҡ„еӯ—з¬ҰдёІеӨ„зҗҶпјҢдҪҶжҳҜзӣёвҪҗдәҺ StringBufferпјҢStringBuilder е…·жңү жӣҙвҫјзҡ„жҖ§иғҪгҖӮ еӣ жӯӨпјҢеҪ“еӯ—з¬ҰдёІеӨ„зҗҶйңҖиҰҒйў‘з№Ғдҝ®ж”№ж—¶пјҢе»әи®®дҪҝвҪӨ StringBuffer жҲ– StringBuilderпјӣвҪҪеҪ“еӯ—з¬ҰдёІеӨ„ зҗҶдёҚйңҖиҰҒдҝ®ж”№ж—¶пјҢеҸҜд»ҘдҪҝвҪӨ String
